# 手机号存在性判断函数 - 实现计划

## 概述
使用 Go 语言实现从本地数据加载手机号并判断是否存在的函数，支持命令行选择方案。

## 约束条件
- 最多 20 亿手机号
- 内存占用 ≤ 2GB
- 必须精确无误，无假阳性
- 实时响应要求高

## 支持方案
- `-t bitmap`: 基于 Roaring Bitmap 实现（当前实现）
- `-t tree`: 预留方案（后续实现）

## 实现步骤

### 步骤 1：初始化 Go 项目并安装依赖
- 创建 `go.mod` 和 `go.sum`
- 安装 Roaring Bitmap 库：`github.com/RoaringBitmap/roaring/roaring64`

### 步骤 2：创建 phone_numbers.txt 测试数据文件
- 创建包含 10-20 个测试手机号的文本文件
- 格式：一行一个手机号，如 `13800138000\n13900139000`
- 使用中国常见的手机号格式

### 步骤 3：实现核心逻辑 phonecheck.go
- 定义接口：
  ```go
  type PhoneChecker interface {
      PhoneExists(phone string) bool
      Close() error
  }
  ```
- 实现 `bitmapChecker` 基于 Roaring Bitmap
- 实现 `treeChecker` 占位（后续完善）
- 实现工厂函数 `NewPhoneChecker(typ string, dataFile string) (PhoneChecker, error)`

### 步骤 4：实现 generator.go 工具
- 读取一行一个手机号的文本文件
- 将手机号转换为 64 位整数
- 生成 Roaring Bitmap 文件（.bin 格式）

### 步骤 5：创建 main.go 入口
- 解析命令行参数 `-t` (类型) 和 `-f` (数据文件)
- 加载数据并进入交互式查询

### 步骤 6：创建 phonecheck_test.go 单元测试
- 测试各方案的正确性

## API 设计
```go
// 创建 PhoneChecker
func NewPhoneChecker(typ string, dataFile string) (PhoneChecker, error)

// 判断手机号是否存在
type PhoneChecker interface {
    PhoneExists(phone string) bool
    Close() error
}
```
